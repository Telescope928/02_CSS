CSS(Cascading Style Sheets)

        마크업언어(html)가 실제로 화면에 표시되는 방법을 기술하는 언어
        (모양, 색상, 크기, 배경이미지, 움직임,...)

        하나의 요소에 여러 CSS 코드를 겹쳐서 원하는 스타일을 만들어냄

        W3C에서 CSS를 웹표준으로 지정함

CSS 선택자 (CSS Selector)

        현재 문서 내에서 특정 요소를 선택하는 문자 또는 문자열(기호)

        * 요소(Element) : 시작태그 + 속성 + 내용 + 종료태그
    **css 코드를 작성하는 방법**
  • head 태그에 style 태그를 작성한 후 내부에 css 코드 작성
  • 원하는 요소에 직접 style 속성을 이용해 css 코드 작성
  • 외부 .css 파일을 생성해 연결하는 방법 (현업에서 가장많이 사용)

태그 선택자

        -현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자
        작성법
        
            태그명{css코드;}

id(#) 선택자

        html 문서 내에서 id 가 일치하는 요소를 선택하는 선택자
작성법 #id속성값{css코드;}
  1. id 선택자 테스트 1
  2. id 선택자 테스트 2
  3. id 선택자 테스트 3

class(.) 선택자

        html 문서 내에서 클래스 속성 값이 일치하는 모든 요소를 선택하는 선택자
        작성법
        .class속성값{css 코드;}
        *** 요소에 css를 적용하기 위한 선택자를 작성할 때
        id 선택자보다 class 선택자를 이용하는 것이 좋다

        왜? id 속성 자체가 css 보다 js 에서 많이 사용되기 때문

모든요소선택자(*, asterRisk)

        단독 사용시 문서에 존재하는 모든 요소 선택자
        다른 선택자와 혼용 사용시 선택된 요소의 모든 자식, 후손, 형제까지 모두 적용
        
        *css 적용 우선순위
        1. 선택자 범위가 가장 작은 선택자
           (!important > 인라인 스타일 > 아이디 > 클래스 > 태그 > 모든요소)
        2. 태그 내에서 같은 속성 값이 있을 경우 가장 마지막에 작성된 클래스
           내에 존재하는 css 속성값이 적용
