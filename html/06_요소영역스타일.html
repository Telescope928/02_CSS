요소의 영역(여백)관련 스타일

        html 요소는 총 4가지의 영역으로 구성되어있음
        1. content(내용) 영역 : 요소의 내용이 작성되어지는 영역
           (시작태그와 종료태그 사이에 작성되는 내용을 생각하면 된다.)
        2. padding : content와 border 사이의 공간
        3. border : 요소의 테두리 영역 content 보다 바깥쪽에서 content를 감싸고 있음
        4. margin(여백, 간격) : 다른 요소와의 간격을 나타내는 영역
    
    -width/height 크기 속성의 범위를 지정하는 class속성 
    1. content box : 가로세로 지정 시 content 영역의 크기만 조정(기본값) 
    2. border box : 가로세로지정 시

padding, margin, border 의 방향성

      (000 == padding, margin, border 를 나타낸다.)

      000 - top : 크기 -> 위쪽 방향 크기 지정
      000 - bottom : 크기 -> 아래쪽 방향 크기 지정
      000 - left : 크기 -> 왼쪽 방향 크기 지정
      000 - right : 크기 -> 오른쪽 방향 크기 지정

      border 의 속성 값은 [크기, 모양, 색상] 형식으로 지정

      padding | margin | border-width | border-radius : 속성 값(크기)
      -> 속성 값을 1개만 줄 때 : 상하좌우 동일한 값 부여
      -> 속성 값을 2개만 줄 때 : 상하 좌우 2가지로 나누어서 부여(button, padding 에 자주 사용)
      -> 3개 : 상 좌우 하 | 위->좌우->아래쪽 순서로 부여
      -> 4개 : 상 우 하 좌 순서대로 부여

margin : auto;

      다른 요소와의 간격을 자동으로 지정하여 가운데 정렬하게 만드는 속성 값
      - 기본적으로(보통) 좌우 간격만 자동으로 지정
      - 지정되는 대상은 block 형식의 요소여야 한다.
        (inline, inline-block, block 은 text-align : center 를 이용해 가운데 정렬)
        (text-align == 글자 정렬)
