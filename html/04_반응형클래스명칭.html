반응형 클래스 명칭 살펴보기

단위 관련

        px : 픽셀. 고정된 크기의 단위
        % : 퍼센트. 부모요소에 비례한 상대단위
            기기 크기에 따라 자동으로 웹사이트 내부에 태그 요소들 크기 조정
        em, rem : 텍스트 크기 기준의 상대 단위
        vw, vh : 뷰포트 너비 높이의 상대 단위(vh의 경우 웹사이트 세로 크기 단위로 많이 사용)

레이아웃 관련

        FlexBox : 1차원 방향으로 정렬할 때 사용하는 레이아웃
        Grid : 2차원 방향으로 정렬
        Container : 콘텐츠를 감싸는 부모 요소로 일반적인
                    고정 폭 또는 최대 폭 지정
        Column, Row : 그리드 시스템에서 열과 행을 의미
                      여러 요소를 정렬 할 때 사용
                      (Column == col)
                      한줄 기준으로 12개의 col이 존재
                      col-6 이라고 작성 할 경우 절반 정도를 의미
                      (Row == row)
        justify-content : center == 가로정렬
        align-item : center == 세로 정렬
        Margin : m-숫자값 or 영어 사용 (m-1(가장작은 단위), m-5 등)
        Padding : margin과 같이 p-1, p-5 사용
        Margin, Padding, Border, Border-radius 는 1~5까지 숫자로 표현가능
